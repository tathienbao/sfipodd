# SFI Podcast Downloader

Một công cụ Python để tải xuống podcast từ SFI (Swedish Radio International).

## Tính năng

- Tải xuống podcast từ SFI
- Quản lý file podcast đã tải
- Giao diện dòng lệnh dễ sử dụng

## Cài đặt

1. Clone repository này:
```bash
git clone <your-repository-url>
cd sfipodd
```

2. Cài đặt các dependencies:
```bash
pip install -r requirements.txt
```

## Sử dụng

Chạy script chính:
```bash
python sfi_podcast_downloader.py
```

## Cấu trúc dự án

```
sfipodd/
```
